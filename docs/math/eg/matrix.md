## 名词

```latex
symmetric % 对称
the transpose of A % A 的转置
an inverse of matrix A % A 的逆
positive definite % 正定
negative definite % 负定
positive semidefinite % 正半定
negative semidefinite % 负半定
indefinite % 不定
identity matrix % 单位矩阵
an eigenvalue of the matrix A % A 的特征值
```

## 表达

```latex
% 矩阵主对角线上的所有元素
all the elements on the leading diagonal of the matrix
% 如果方阵有逆，则称为可逆矩阵或非奇异矩阵。否则，则称其为奇异矩阵
If a square matrix has an inverse it is called invertible or nonsingular. Otherwise, it is called singular.
% 向量 x 被称为与特征值 λ 相关的特征向量
The vector x is called an eigenvector associated to the eigenvalue λ.
% 如果一个矩阵的所有特征值都位于一个不包含零的紧区间内，则称该矩阵是有界矩阵
A matrix is said to be bounded if its eigenvalues lie in the compact interval that does not contain zero
```
